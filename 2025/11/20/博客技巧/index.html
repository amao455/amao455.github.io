博客技巧

hexo相关指令

init

hexo init [folder]

新建一个网站。如果没有设置folder，hexo会默认在当前的文件夹中建立网站。

new

hexo new [layout]

新建一篇文章。

选项 描述
-p, –path 文章的路径。自定义文章的路径
-r, –replace 如果存在的话。替换当前的文章
-s, –slug 文章别名。自定义文章的URL

generate

hexo generate

生成静态文件。

publish

hexo publish [layout]

发表草稿。

server

hexo server

启动服务器。

选项 描述
-p, –port 重设端口
-s, –static 只使用静态文件
-l, –log 启用日志

deploy

hexo deploy

部署网站。

选项 描述
-g, –generate 在部署前生成

render

hexo render [file2]…

渲染文件

migrate

hexo migrate

从其他博客系统迁移内容。

clean

hexo clean

清除缓存文件和已生成的静态文件

list

hexo list

列出所有路由。

version

hexo version

显示版本信息。

config

hexo config [key] [value]

列出网站的配置。
